# Lightcone Research — Project Governance

**Version:** 0.1 (Provisional) | **Last updated:** 2026-03-01

Governance for the Lightcone Research project and its repositories (ASP, Prism,
Canvas, and related tools). Intentionally minimal — will evolve as the community grows.

## Project Leads

**Liam** (UC Berkeley) and **Francois** (CNRS). Set direction, hold final decision
authority, manage the GitHub org, and enforce the Code of Conduct. If a lead steps
down, the remaining lead assumes sole authority.

## Decision Making

The project leads have sole discretion over all decisions at this stage.
Contributors can propose changes via GitHub issues or pull requests.

## Contributions

See [CONTRIBUTING.md](CONTRIBUTING.md). All contributions are made under Apache-2.0
with [DCO sign-off](https://developercertificate.org/).

## Code of Conduct

All participants must follow the [Code of Conduct](CODE_OF_CONDUCT.md). Contact
the project leads for enforcement: lhparker@berkeley.edu, francois.lanusse@cnrs.fr.

## Future Evolution

This structure reflects the project's early stage and will be revisited.

The project's open-source nature and Apache-2.0 license are not contingent on any
particular organizational structure.

Governance changes can be proposed via PR and are adopted by consensus of the
project leads.
